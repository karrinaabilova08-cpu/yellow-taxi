# Yellow-taxi
Проект был создан для анализа yellow taxi в Нью-Йорке для экзамена по программированию.

# Бизнес-вопросы, на которые я ответила
1.  Когда (в какое время суток) самый большой спрос на такси?
2.  Какие дни недели являются пиковыми по количеству поездок?
3.  Как меняется средняя стоимость поездки в зависимости от времени суток?
4.  Как меняется длительность поездки по часам суток?
5.  Как меняется скорость поездки по часам суток?
